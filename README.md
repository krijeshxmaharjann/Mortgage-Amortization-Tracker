# 🏠 Mortgage Amortization Tracker

An interactive mortgage analytics dashboard built with **Python and Streamlit** that helps users analyze mortgage payments, loan payoff strategies, amortization schedules, and home affordability.

This project demonstrates **object-oriented programming, financial modeling, data analysis, and interactive visualization**.

---

# 🚀 Features

## 📊 Mortgage Amortization Engine

- Calculates monthly mortgage payments
- Generates complete month-by-month amortization schedules
- Tracks:
  - Principal payments
  - Interest payments
  - Remaining loan balance
  - Total interest paid
  - Total principal paid

---

## 💰 Extra Payment Payoff Analysis

Users can add extra monthly principal payments to analyze:

- Early loan payoff
- Years saved
- Interest savings
- Total cost reduction

---

## 🏦 Loan Comparison

Compare mortgage scenarios based on:

- Loan amount
- Interest rate
- Loan term
- Monthly payments
- Total interest cost
- Total loan cost

---

## 🏡 Mortgage Affordability Calculator

Estimates:

- Maximum affordable monthly payment
- Maximum loan amount
- Estimated home price
- Debt-to-income ratio

Uses common mortgage affordability calculations.

---

## 📈 Interactive Financial Visualizations

Built with Plotly:

- Remaining loan balance chart
- Principal vs interest breakdown
- Monthly payment breakdown
- Cumulative interest paid
- Cumulative principal paid

---

## 📥 Export Data

Users can:

- Download amortization schedules as CSV files
- Analyze mortgage data externally
